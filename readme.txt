Easily write Urdu unicode on WordPress without installing Urdu keyboard on your system.

== Description ==

This is the First Urdu writing keyboard Plugin for WordPress. Now you can easily write Urdu posts and pages directly in WordPress without installing Urdu keyboard on your system.

This Urdu Keyoboard (Phonetic) for WordPress, enables you to easily type Urdu in WordPress without installing Urduphonetic keyboard in your system.

== Installation ==

1. Unzip the download package
2. Upload `urdu-keyboard` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress

= Manual Plugin Installation =
1. Download Urdu Keyboard Plugin to your desktop.
2. If downloaded as a zip archive, extract the Plugin folder to your desktop.
3. With your FTP program, upload the Plugin folder to the `wp-content/plugins` folder in your WordPress directory online.
4. Go to Plugins screen and find Urdu Keyboard Plugin in the list.
5. Click Activate Plugin to activate it.

== Frequently Asked Questions ==

= How to use Urdu Keyboard in WordPress =

Simple and easy same as writing the posts or pages in WordPress. When plugin is enabled you can write Urdu text in post/page title and content using your system keyboard.

Keyboard layout help available in posts screen and as well as option page.

= How to change keyboard language? =

Pressing `Ctrl + Space` to change keyboard writing from `English to Urdu` or `Urdu to English`.
